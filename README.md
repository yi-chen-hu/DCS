NCTU_DCS
===
這門課是交大電子所張添烜教授開的數位電路與系統

整學期共有 
- 1 Project 
- 5 HWs
- 9 Labs
- 1 Online Test

都是用 system verilog 實現特定演算法

測資則沒有規定必須使用何種程式語言

我個人產生測資是使用 C++，再用 pattern.v 去讀取 txt 檔

因為課程規劃的關係，前期助教都會提供 pattern，所以有些 HW 或 Lab 沒有測資，只有 design
